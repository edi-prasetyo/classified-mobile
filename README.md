## About the App

Graha Classified is a professional, cross-platform mobile application built with Flutter, designed as a comprehensive marketplace for buying and selling goods through classified ads. This app serves as an excellent portfolio piece, demonstrating advanced mobile development skills, clean architecture, and modern UI/UX design. It's fully functional for both Android and iOS, ready for customization and deployment to real-world clients.

### Key Features

- **User Authentication**: Secure login and registration with data persistence
- **Browse Categories**: Intuitive category-based browsing for easy navigation
- **Detailed Listings**: Rich product details with images and descriptions
- **Favorites**: User-friendly favorites system for saving preferred items
- **Profile Management**: Complete user profile and listing management

### Technologies Used

- **Flutter**: Cross-platform framework for native Android and iOS apps
- **Dart**: Programming language for robust app logic
- **Riverpod**: State management for scalable architecture
- **Dio**: HTTP client for API integrations
- **Cached Network Image**: Efficient image loading and caching
- **Flutter Secure Storage**: Secure data storage for sensitive information
- **Flutter SVG & HTML**: Rich content rendering
